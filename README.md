# TicketHub Frontend

This is the Next.js frontend for TicketHub.  
It allows users to submit support tickets to the serverless backend.

## Live Demo
https://tickethub-aws-ui.vercel.app/

## Tech Stack

- Next.js 15
- React 19
- TailwindCSS

## Running Locally

```bash
git clone https://github.com/romelt777/tickethub-aws-ui.git
cd tickethub-aws-ui
npm install
npm run dev
```
